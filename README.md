# 조언 생성기

Advice Slip API에서 뉴스정보를 가져와 표시하는 간단한 웹 애플리케이션입니다.

## 기능

*   최근뉴스를 표시합니다.
*   카테고리별 뉴스를 표시합니다.
*   뉴스를 검색할 수 있습니다.

## 사용 기술

*   HTML
*   CSS
*   JavaScript (Fetch API)

## 실행 방법

1.  저장소를 클론합니다.
2.  웹 브라우저에서 `index.html`을 엽니다.

## 사용된 API

*   Advice Slip API: `https://api.adviceslip.com/`
