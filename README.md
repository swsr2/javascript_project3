# 조언 생성기

Advice Slip API에서 무작위 조언을 가져와 표시하는 간단한 웹 애플리케이션입니다.

## 기능

*   무작위 조언을 표시합니다.
*   조언 ID를 표시합니다.
*   새로운 조언을 생성하는 버튼이 있습니다.

## 사용 기술

*   HTML
*   CSS
*   JavaScript (Fetch API)

## 실행 방법

1.  저장소를 클론합니다.
2.  웹 브라우저에서 `index.html`을 엽니다.

## 사용된 API

*   Advice Slip API: `https://api.adviceslip.com/`
